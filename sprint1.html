Css Selectors

In CSS gebruik je selectors om te bepalen welke HTML-elementen je wilt stylen. Selectors zijn als "aanwijzers" die je richt op specifieke delen van je webpagina, zoals kopteksten, paragrafen, of knoppen. Met de juiste selector kun je een element een eigen stijl geven, zoals een kleur, lettertype of grootte.

ID Selector (#idNaam)

De ID selector gebruik je om één specifiek element op de pagina te stylen. Een ID is uniek, dus het wordt maar één keer per pagina gebruikt. De ID selector begint met een # gevolgd door de ID-naam.

#special-quote

Class Selector (.classNaam)

De class selector gebruik je om meerdere elementen dezelfde stijl te geven. Dit is handig voor herbruikbare stijlen. Een class selector begint met een . gevolgd door de class-naam.

.quote-container

Element Selector (element)

De element selector gebruik je om alle elementen van hetzelfde type te stylen, zoals alle paragrafen (p) of kopteksten (h2).

P

HTML zegt wat er op de pagina staat (zoals tekst of afbeeldingen). CSS-selectors helpen je om precies te kiezen welke delen van de pagina je wilt stylen, zodat ze er beter uitzien.
